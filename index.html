scomm.ai Public Endpoint

This domain is used for OAuth redirects and mobile app link verification for the scomm.ai communication platform.

Well-Known Resources

These files allow Android and iOS devices to securely verify ownership of this domain and route HTTPS links into the app.

  • /.well-known/assetlinks.json
    Android App Links verification
  • /.well-known/apple-app-site-association
    Apple Universal Links verification

OAuth Redirect Endpoints

The following paths are used as OAuth redirect URIs:

  • /oauth2/google
  • /oauth2/microsoft
  • /oauth2/yahoo
  • /oauth2/imap

When opened on a supported device, these URLs automatically launch the scomm application using secure deep linking.

If you reached this page manually, you probably meant to open the scomm application.

Learn more at: https://scomm.ai
