Delete Account

Learn how to delete your scomm.ai account and manage your data

DATA DELETION PROCESS

How to Delete Your Account

  • Method 1: Through Client Application
  • Open application settings
  • Navigate to Account section
  • Select "Delete Account"
  • Confirm deletion
  • Data deletion begins immediately
  • Method 2: Email Request
  • Email: privacy@scomm.ai
  • Subject: Account Deletion Request
  • Include: Email address associated with account

Response within 5 business days

What Gets Deleted

Immediately Deleted:

  • Account credentials
  • OAuth tokens
  • Active sessions
  • Device identifiers
  • Access Control Lists
  • Using Party associations

Authentication tokens

Deleted Within 30 Days:

  • Public keys
  • Email address hashes
  • Encrypted private key backups
  • User preferences

Application settings

Deleted Within 90 Days:

  • Backup copies

Cached data

What Cannot Be Deleted (Legal Requirements)

Retained for 7 Years:

  • Billing transaction records (tax/accounting law)
  • Invoices
  • Payment history

Relay usage data for billing purposes

Anonymized:

  • After retention period, billing records are anonymized
  • Personal identifiers removed

Retained only as aggregate statistics

Partial Deletions

Delete Public Keys Only:

  • Account settings → Encryption → Delete Public Keys
  • Or email privacy@scomm.ai with "Delete Public Keys" request

Account remains active

Delete Encrypted Private Key Backup:

  • Account settings → Encryption → Delete Cloud Backup

Local private keys on your devices remain unaffected

Remove Using Parties:

  • Paying Party can remove Using Parties

Using Party data deleted within 30 days

Remove Devices:

  • Account settings → Devices → Remove Device

Device identifier deleted immediately

Revoke OAuth Permissions:

Google
Google Account → Security → Third-party apps
  • Microsoft: Microsoft Account → Apps and services

Revokes email access, account remains active

Account Closure Without Deletion

Deactivate Account:

  • Stops billing
  • Retains data for potential reactivation
  • Can be reactivated within 2 years

After 2 years, treated as inactive and deleted

Multi-User Account Deletions

Paying Party Deletes Account:

  • All Using Parties are notified
  • Using Parties lose access

All associated data deleted per schedule above

Using Party Deletes Account:

  • Only that Using Party's data deleted

Paying Party and other Using Parties unaffected

Service Dependency Certain features of our services may not function properly if required data is deleted, restricted, or if permissions are revoked.
